Alat & Bahan yang Dibutuhkan

Siapkan semua komponen berikut sebelum memulai praktik. Pastikan semua alat dalam kondisi baik dan kabel tidak putus.

ESP32

Mikrokontroler utama dengan WiFi & Bluetooth terintegrasi. Gunakan kabel USB untuk menghubungkan ke komputer.

LED

LED kecil yang dapat diaktifkan dan dimatikan melalui mikrokontroler. Sambungkan ke pin digital pada ESP32.

Servo Motor

Servo mini dengan sudut putar 0°–180°. Sambungkan ke pin PWM pada ESP32.

Sensor Ultrasonik

Sensor pengukur jarak menggunakan gelombang suara. Memiliki 4 pin: VCC, GND, Trig, Echo.

Setting ESP32 di Windows dengan PictoBlox

1. Install Driver ESP32

Install driver sesuai chip USB:

  • CP2102Download
  • CH340
Tujuan: agar ESP32 terbaca sebagai COM Port di Windows

2. Cek Port COM

  1. Hubungkan ESP32 ke laptop
  2. Buka Device Manager
  3. Lihat bagian Ports (COM & LPT)

Contoh:

USB to UART Bridge (COM3)

3. Install PictoBlox

Download dan install PictoBlox dari website resminya.

4. Tambahkan Board ESP32

  1. Buka PictoBlox
  2. Klik Board → Add Extension
  3. Cari ESP32
  4. Klik Add

5. Setting Board dan Port

  • Board: ESP32
  • Port: sesuai COM (contoh COM3)

6. Gunakan Upload Mode

Pilih Upload Mode agar program tersimpan di ESP32.

7. Test LED

Gunakan GPIO 2 untuk LED built-in (umumnya pada ESP32).

8. Jika Gagal Upload

  • Tekan tombol BOOT saat upload
  • Gunakan kabel data
  • Pastikan driver sudah terinstall

Tips

  • Gunakan kabel pendek
  • Tutup aplikasi lain seperti Arduino
  • Pastikan board tidak rusak

WIRING DAN PINOUT

Konfigurasi Pin (Disarankan)

  • LED 1 → GPIO 19
  • LED 2 → GPIO 21
  • Ultrasonik Sensor
    • TRIG → GPIO 5
    • ECHO → GPIO 18
  • Servo → GPIO 25

Materi

Ikuti langkah-langkah berikut untuk menyelesaikan praktik. Pastikan setiap langkah dilakukan dengan benar untuk hasil yang optimal.

1. Kontrol LED

Pada tahap pertama, kamu akan belajar menyalakan dan mematikan LED menggunakan perintah blok di PictoBlox. Coba buat LED berkedip setiap 1 detik sebagai latihan dasar.

Pelajari

2. Kontrol Servo

Sambungkan servo ke pin PWM gpio 25 ESP32. Buat program untuk memutar servo ke sudut 0°, 90°, dan 180° dengan jeda 2 detik.

Pelajari

3. Sensor Ultrasonik

Hubungkan sensor ultrasonik ke pin VCC, GND, Trig (gpio 5), dan Echo (gpio 18) ESP32. Buat program untuk mengukur jarak dan tampilkan hasilnya di serial monitor.

Pelajari

4. Bluetooth

Pilih ekstensi ESP32 Bluetooth pada menu ekstensi PictoBlox.Hubungkan smartphone ke ESP32 melalui pengaturan Bluetooth. Cari nama "ESP32-BT". Gunakan aplikasi Bluetooth Terminal, kirim karakter seperti "1" untuk nyalakan LED dan "0" untuk mematikan.
